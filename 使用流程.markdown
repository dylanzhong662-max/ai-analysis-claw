## 使用流程

**使用流程**                                                                 

​                                                                      

 **第一步：生成分析提示词**                                                          

   

 在项目目录下运行脚本：

```                                                          
\# 只生成提示词文件（默认）

 python3 gold_analysis.py



 \# 调用默认 Claude 模型

 python3 gold_analysis.py --api



 \# 调用 DeepSeek R1（推理模型）

 python3 gold_analysis.py --api --model deepseek-reasoner



 \# 调用 DeepSeek Chat

 python3 gold_analysis.py --api --model deepseek-chat



 \# 指定其他 Claude 模型

 python3 gold_analysis.py --api --model claude-opus-4-6
```

Btc                                                                                                                                                       

``` 
python3 btc_analysis.py --api --model deepseek-reasoner
```

股票:

```
python3 tech_stock_analysis.py --ticker GOOGL --api --model deepseek-reasoner

python tech_stock_analysis.py --ticker NVDA --api --model           
      deepseek-reasoner
```



脚本会自动完成以下工作：                                                         

 \- 从 Yahoo Finance 拉取黄金日线/周线数据                                                 

 \- 获取 DXY、10年期美债收益率、VIX、白银 4个宏观数据源                                          

 \- 计算所有技术指标（MACD、RSI、ADX、Stochastic、布林带、OBV 等）

 \- 读取 backtest_results/performance.csv，注入历史胜率反馈                                        

 \- 将完整分析请求保存到 gold_prompt_output.txt，并打印到终端                                       

​                                                                      

 \---                                                                   

 **第二步：把提示词发给 Claude**                                                       

​                                                                      

 打开 gold_prompt_output.txt，把全部内容复制，粘贴到 Claude.ai 对话框发送。                                

​                                                                      

 你当前所在的这个对话框（Claude Code）也可以直接分析，就像本次对话开头你做的那样。                            

​                                                                      

 \---                                                                   

 **第三步：把 Claude 返回的 JSON 存档**             

​                                                                      

 Claude 会返回一个 JSON 信号。把它保存下来，后续回测需要用到。

​                                                                      

 \---                                                                   

 **回测完成后（已有 signals.csv）**                                                      

​                                                                      

 每次新的回测跑完后，performance.csv 会自动更新。**下次再运行** **python3 gold_analysis.py** 

 **时，脚本会自动读取最新胜率并注入到提示词里**，不需要手动操作。                                       

​                               

 终端会显示类似：                                                             

​                               

 已载入回测指标 — 胜率: 20.7% 盈利因子: 1.56 连续亏损: 1 次                                       

​                                                                      

 如果胜率低于 40% 或连续亏损 ≥ 2 次，提示词里会自动出现对应的警告，Claude 会据此收紧开仓标准
